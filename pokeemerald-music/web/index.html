Pokémon Emerald — overworld BGM

decoded from the pokeemerald source (MIDI + m4a voicegroups + DirectSound samples), synthesized live in the browser by an AudioWorklet reimplementing the GBA m4a sound engine.

play uses the game's own 8-bit samples; sf2 swaps in the SC-88-quality Roland instruments + Charm drums from the GBApokemonTestLite soundfont (same sequence, same engine — first use fetches ~24 MB). Songs loop like on the GBA (loop points come from the [/] markers in the game's MIDI files). See README.md one directory up for how the engine works and how extraction was done.
